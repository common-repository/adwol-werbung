Nutze dieses Plugin, um deinen Blog zu monetarisieren. Mit dieser Erweiterung kannst du Anzeigen mit wenigen Klick in deine Seite intergrieren.

== Description ==

*Binde mit diesem Plugin ganz einfach Anzeigen in Artikel und Seiten ein.*

Erstelle dir unter https://adwol.com einen kostenlosen Account, um mit Werbung Geld zu verdienen. Nach der Einrichtung, kannst du weitere Einstellungen im Backend vornehmen.

<span style=\"text-decoration: underline;\">Die wichtigsten Funktionen im Überblick:</span>

*   Verdiene mit deinem Content Geld
*   meistens kontextrelevante Anzeigen
*   einfache Einrichtung
*   Alternative zu Google AdSense, Plista und Co.
*   Einbindung als Widget, im Artikeltext und als Artikelempfehlung (ähnliche Beiträge)

== Installation ==

1. Installiere das Plugin im Admin-Bereich oder extrahiere die ZIP-Datei, lade sie in den Ordner wp-content/plugins/ hoch und aktiviere das Plugin im Admin-Bereich.
2. Ergänze die fehlenden Angaben unter Einstellungen -> AdWol Werbung.

== Screenshots ==
1. Artikelempfehlungen: Binden Sie die Artikelempfehlungen ein, um mehr Traffic innerhalb Ihrer Seite zu erhalten. Außerdem wird eine native Anzeige mit eingeschaltet.
2. InPost Anzeigen: Lassen Sie innerhalb Ihrer Artikel automatisch responsive Anzeigen erscheinen.
3. Werbe Widget: Sie können ein Werbewidget einfach per Drag and Drop zu Ihrer Seitenleiste, Fußzeile etc. hinzufügen. Dann werden automatisch möglichst relevante Banner oder Textanzeigen eingeblendet!
